# TableView
tableview是一款自定义表格控件，使用方便，自定义属性

#属性
属性名称	意义	默认值
rownum	行数	0
colnum	列数	0
head_is_show	是否显示表格表头	false
headtext	表头文字	默认文字
headheight	表头高度	80dp
tablehead_height	表格标题高度	100dp
tablecontent_height	表格内容高度	100dp
